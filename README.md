# Clearwater & Sons — Demo Site

> Portfolio demonstration project. All content is fictional, no real client data was used.

## Overview
A fully responsive website for a fictional plumbing business, built to showcase frontend development skills.

## Features
- Responsive design (mobile & desktop)
- Mobile hamburger menu with animated icon toggle
- Sticky header with scroll effect
- Scroll reveal animations (Intersection Observer API)
- Smooth scroll with fixed header offset
- Back to top button
- Customer reviews section
- Contact section with embedded Google Map
- Structured Data (Schema.org) for SEO
- Back to top button

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome 6 (icons)
- Google Fonts (Inter, Outfit)

## Notes
This is a demo project built for portfolio purposes. No real client data was used.
